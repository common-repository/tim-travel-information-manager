TIM is a Cloud-based Web application, which allows managing the full operation of travel agencies.
 
== Description ==
 
TIM is a Cloud-based Web application, which allows managing the full operation of travel agencies, tour operators, hotels and transport companies in the tourism sector.

This plugin connects you wordpress site with your TIM account.
 
Major features in Tim Travel Manager include:

* Travel itineraries
* Online bookings
* Ease configuration and customization
* SEO ready
* Online availability
* Shopping cart funcionality
* Full integration with payment systems, including Paypal, BAC, Banco de Costa Rica an others

== Installation ==
 
Upload the Tim Travel Manager plugin to your website, Activate it, then enter your Api credentials
 
Steps:
 
1. Upload the plugin to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enter yur TIM account API
That's it
